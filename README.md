## ✨ Features

- 🎯 Immersive 3D hero section with animated particles and neon rings

- 🧠 Backend-focused professional profile section
- 🏦 Fintech and core banking experience timeline
- 🛠️ Skills grouped by frontend, backend, tools, and databases
- 📂 Project cards with impact-focused summaries
- 📬 Contact form with mail client integration

## 🏁 Getting Started

Clone the repository:

```bash
git clone https://github.com/rahulkumar384/Rahul-s_Portfolio.git
```

Move into the project folder:

```bash
cd Rahul-s_Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open locally:

```text
http://127.0.0.1:5173
```

## 🚀 Deploy on Vercel

This project includes a `vercel.json` file, so it is ready for Vercel deployment.

Recommended Vercel settings:

| Setting | Value |
| --- | --- |
| Framework Preset | `Vite` |
| Install Command | `npm install` |
| Build Command | `npm run build` |
| Output Directory | `dist` |

Deployment flow:

1. Push the repository to GitHub.
2. Open [Vercel](https://vercel.com/).
3. Click **Add New Project**.
4. Import this GitHub repository.
5. Deploy with the default Vite settings.
